## Requirements

Before installing Viber3D, make sure you have:

- Node.js 16.x or later
- A modern web browser with WebGL support
- npm, yarn, or pnpm package manager

## Installation

Run the script to create a new project:

```bash [Terminal]
npx viber3d@latest init
```
